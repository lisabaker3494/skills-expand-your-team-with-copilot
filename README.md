## Social Sharing

Each activity card includes **Share** buttons so students and teachers can spread the word with friends. Supported platforms:

- **Facebook** — shares a link to the activity
- **X (Twitter)** — posts a pre-filled tweet with the activity name and link
- **LinkedIn** — shares the activity URL on LinkedIn
- **WhatsApp** — sends a message with the activity name and link
- **Email** — opens a pre-filled email with subject and body

Share links are built using the current page URL with the activity name appended as a query parameter (`?activity=<name>`), so recipients land directly in context.
